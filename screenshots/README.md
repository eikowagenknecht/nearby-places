# Screenshots

This folder contains example screenshots of the generated HTML views.

## To update screenshots:

1. Open each HTML file in your browser:
   - `list.html` - Condensed list view
   - `overview.html` - Detailed overview
   - `map.html` - Interactive map

2. Take screenshots and save them as:
   - `list-view.png`
   - `overview-view.png`
   - `map-view.png`

3. Recommended screenshot settings:
   - Browser window: ~1200px wide
   - Zoom: 100%
   - Include a few items in the view
